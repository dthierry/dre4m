DRE4M

Introduction

DRE4M or dre4m is a framework for the planing of the technological makeup of a generic sector. The current incarnation framework coalesces into a linear program (LP). The program was laid out using JuMP, which also serves as the interface between model and solver. dre4m is subject to the terms of the MIT license.

Installation

At the Julia REPL press ] mode, then dre4m can be obtained directly from the repository as follows,

add https://github.com/dthierry/dre4m.git

The dre4m layout

The source is organized as follows,
